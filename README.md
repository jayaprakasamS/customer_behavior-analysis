## Project Workflow

### 1. Data Preparation in Python

#### Data Loading
The dataset was loaded using **Pandas** for analysis.

#### Initial Data Exploration
Basic exploration was performed using:

```python
df.info()
df.describe()
```

These commands help understand the dataset structure and summary statistics.

---

#### Missing Value Handling
Missing values in the **`review_rating`** column were handled by imputing the **median rating for each product category**.

---

#### Column Standardization
All column names were converted to **snake_case** to improve readability and maintain consistency.

---

#### Feature Engineering
Two new features were created:

- **age_group** → created by binning customer ages  
- **purchase_frequency_days** → derived from purchase data  

---

#### Data Consistency Check
It was identified that **`discount_applied`** and **`promo_code_used`** contained redundant information.

Therefore:

- The **`promo_code_used`** column was removed.

---

#### Database Integration
The cleaned dataset was connected to **PostgreSQL** and loaded into the database for further SQL analysis.

---

## SQL Data Analysis

SQL queries were used to answer important **business questions**.

### Key Analyses Performed

**Revenue by Gender**  
Compared total revenue generated by **male vs female customers**.

**High Spending Discount Users**  
Identified customers who used discounts but still spent **above the average purchase amount**.

**Top 5 Products by Rating**  
Found products with the **highest average review ratings**.

**Shipping Type Comparison**  
Compared average purchase amounts between **Standard** and **Express shipping**.

**Subscribers vs Non-Subscribers**  
Compared **revenue and spending patterns** of subscribed and non-subscribed customers.

**Discount Dependent Products**  
Identified products with the **highest percentage of discounted purchases**.

**Customer Segmentation**

Customers were segmented into:

- New Customers  
- Returning Customers  
- Loyal Customers  

based on purchase history.

**Top Products per Category**  
Identified the **Top 3 products in each category**.

**Repeat Buyers & Subscriptions**  
Analyzed whether customers with **more than 5 purchases** are more likely to subscribe.

**Revenue by Age Group**  
Determined which **age groups contribute the most revenue**.

---
